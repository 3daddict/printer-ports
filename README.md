# Webpack 4 Boilerplate
## A boilerplate using Webpack 4

## Usage

Install [nodemon](https://github.com/remy/nodemon) globally

```
npm i nodemon -g
```

Install server and client dependencies

```
cd server
npm install

cd client
npm install
```

To start the server
cd server
nodemon
```
To start the client
cd client
npm start
```
